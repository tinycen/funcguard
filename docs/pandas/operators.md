# 支持的运算符

在条件查询中支持以下运算符：

| 运算符 | 说明 |
|--------|------|
| `==` | 等于 |
| `!=` | 不等于 |
| `>` | 大于 |
| `>=` | 大于等于 |
| `<` | 小于 |
| `<=` | 小于等于 |
| `in` | 包含在列表中 |
| `not in` | 不包含在列表中 |
| `null` | 为空（仅判断 NaN/None） |
| `not null` | 不为空（仅判断 NaN/None） |
| `empty` | 为空（同时覆盖 NaN/None 和空字符串 ""） |
| `not empty` | 不为空（同时覆盖 NaN/None 和空字符串 ""） |
| `startswith` | 以指定字符串开头 |
| `endswith` | 以指定字符串结尾 |
| `contains` | 包含字符串 |
| `not contains` | 不包含字符串 |
