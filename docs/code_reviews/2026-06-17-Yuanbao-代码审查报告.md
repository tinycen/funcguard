### 问题 1：裸 except 子句吞掉所有异常，掩盖真实错误

**文件**：`funcguard/ip_utils.py`
**行号**：第 41、58、73 行
**级别**：🔴 严重bug

**问题描述**：
`get_public_ip()` 和 `is_valid_ip()` 函数中使用了裸 `except:` 子句（bare except clause），这会捕获所有异常（包括 `SystemExit`、`KeyboardInterrupt` 等非 Exception 类型的异常），并静默忽略它们。这会导致真实错误被掩盖，难以排查问题。

**代码片段**：
```python
# ip_utils.py 第 41 行
for service in ip_services:
    try:
        response = requests.get(service, timeout=5)
        if response.status_code == 200:
            ip = response.text.strip()
            if is_valid_ip(ip):
                return ip
    except:
        continue   # ← 裸 except，吞掉所有异常

# ip_utils.py 第 58-73 行
def is_valid_ip(ip_string):
    try:
        # ...
    except:
        return False   # ← 裸 except，静默失败
```

**修复建议**：
```python
# 改为捕获特定异常
for service in ip_services:
    try:
        response = requests.get(service, timeout=5)
        if response.status_code == 200:
            ip = response.text.strip()
            if is_valid_ip(ip):
                return ip
    except requests.RequestException:
        continue

# is_valid_ip 中改为
def is_valid_ip(ip_string):
    try:
        parts = ip_string.split('.')
        if len(parts) != 4:
            return False
        for part in parts:
            if not part.isdigit():
                return False
            num = int(part)
            if num < 0 or num > 255:
                return False
        return True
    except (ValueError, TypeError, AttributeError):
        return False
```

### 问题 5：`DataFrameStatistics` 中的掩码缓存不线程安全

**文件**：`funcguard/pd_utils/statistics/df_statistics.py`
**行号**：第 28-31 行
**级别**：🟠 注意问题

**问题描述**：
`DataFrameStatistics` 类的 `_true_mask` 和 `_false_mask` 是可变对象，被缓存并在方法调用中被修改（如通过 `&=` 和 `|=` 操作）。如果在多线程环境中共享同一个实例，掩码缓存可能被并发修改，导致数据竞争。

**代码片段**：
```python
def build_base_mask(self, conditions, logic, true_mask=None, false_mask=None):
    if true_mask is None:
        true_mask = self._true_mask   # ← 返回可变引用，非线程安全
    if false_mask is None:
        false_mask = self._false_mask
    return _original_build_base_mask(self._df, conditions, logic, true_mask, false_mask)
```

**修复建议**：
返回掩码的副本以避免外部修改影响缓存：
```python
def build_base_mask(self, conditions, logic, true_mask=None, false_mask=None):
    if true_mask is None:
        true_mask = self._true_mask.copy()  # ← 返回副本
    if false_mask is None:
        false_mask = self._false_mask.copy()
    return _original_build_base_mask(self._df, conditions, logic, true_mask, false_mask)
```
